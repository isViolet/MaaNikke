# MAA NIKKE

## 为什么会有 MAA NIKKE？

> 本项目的萌芽，源自本人很懒，对于 Nikke 的日常任务表示无法接受，机缘巧合中看到了 MAA 开源了框架，于是我开始了开发 MAA Nikke 之路。

1. 如果你受不了 nikke 每日重复性极高的操作 ~~（虽然也不是很花时间）~~
2. 如果你的时间很宝贵，没有多余精力去完成每日
3. 如果你想体验下自动化游戏的乐趣

> 该程序开发调试的工具皆为本人手机（1080x2400），开发初衷为自用，没有对其他分辨率的机型的调试。
